Code erreur EBICS

091117

EBICS_ACCOUNT_AUTHORISATION_FAILED

Bloquant Authentification EBICS 2.5EBICS 3.0
Description

Le compte bancaire ciblé par la remise n'est pas autorisé dans le contrat EBICS.

Causes fréquentes
  • IBAN du compte débiteur absent de la liste des comptes autorisés dans le paramétrage EBICS
  • Compte ajouté récemment mais contrat EBICS non mis à jour
  • Mauvais IBAN dans le fichier de paiement
Contenu réservé à l'outil Résolution complète dans l'outil →
Action recommandée

Vérifier avec la banque que le compte est bien référencé dans le contrat EBICS. Demander une mise à jour du paramétrage si nécessaire.
