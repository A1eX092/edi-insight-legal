Prise en main rapide

Qu'est-ce qu'EDI Insight ?

Une application web spécialisée dans les flux de paiement bancaires : virements et prélèvements SEPA (ISO 20022), comptes rendus (pacs.002, pain.002, camt), erreurs EBICS et messages EDIFACT. Elle sert à vérifier la conformité d'un fichier, comprendre un rejet, et préparer/convertir des fichiers — sans installation.

Vos fichiers restent chez vous

L'analyse, la validation et la conversion s'effectuent uniquement dans votre navigateur. Le contenu de vos fichiers (montants, IBAN, noms de tiers…) n'est jamais transmis, stocké ni consulté sur nos serveurs — un point clé en environnement bancaire sensible.

Les 5 usages clés

  1. Valider un fichier XML contre les rulebooks SEPA (SCT, SCT Inst, SDD Core/B2B…).
  2. Diagnostiquer un rejet ou lire un relevé (camt) pour comprendre ce qui coince.
  3. Convertir des adresses non structurées vers le format structuré ISO 20022 (échéance nov. 2026).
  4. Générer des fichiers de test conformes (SCT, SDD, ICT, XCT).
  5. Consulter les référentiels : erreurs EBICS 900xx, motifs de rejet (AC01, MS03…), balises XML.

Démarrer en 4 étapes

  1. Ouvrez app.ediinsight.app — aucune installation.
  2. Utilisez librement les fonctions gratuites : validation de conformité (l'export PDF du rapport, lui, est Pro), générateur de fichiers de test, référentiels, et les significations des codes EBICS et des motifs de rejet.
  3. Pour les fonctions Pro — Diagnostiquer, Pourquoi (contraintes bancaires), résolutions détaillées (EBICS & rejets), convertisseur d'adresses et export PDF — connectez-vous : saisissez votre email, recevez un code et tapez-le dans l'app.
  4. Glissez-déposez votre fichier : le résultat s'affiche instantanément.

Guide complet — pas à pas

Chaque fonctionnalité détaillée étape par étape. Les mentions Gratuit et Pro indiquent l'accès requis. Repères : la barre supérieure contient les onglets Accueil, Analyser, Référentiels, EBICS, Plus, et vos initiales (compte) en haut à droite.

Se connecter Gratuit

Connexion sans mot de passe, par code à usage unique — vous restez dans l'app.

  1. Ouvrez une fonction Pro (ex. Analyser → Diagnostiquer) : l'écran de connexion s'affiche.
  2. Saisissez votre email, cliquez « Recevoir un code de connexion ».
  3. Ouvrez l'email reçu : il contient un code (6 à 8 chiffres).
  4. Tapez le code dans l'app, cliquez « Se connecter ».
  5. Vous êtes connecté, sur la même page. Vos initiales apparaissent en haut à droite.
Le code remplace tout lien cliquable — insensible aux filtres anti-spam qui « pré-ouvrent » les liens. Il expire au bout d'1 heure ; au besoin, redemandez-en un.

Valider Gratuit Export PDF : Pro

Contrôle la conformité d'un fichier XML contre les rulebooks officiels.

  1. Onglet Analyser → sous-onglet Valider.
  2. Choisissez le service attendu dans le sélecteur Rulebook (SCT, SCT Inst, SDD Core, SDD B2B…) : le rulebook correspondant — c'est-à-dire le référentiel de règles SEPA propre à ce type de flux — est alors appliqué automatiquement au contrôle.
  3. Cliquez « Importer un fichier XML » (ou glissez-déposez le fichier).
  4. Lisez le résultat : soit « conforme / Aucune anomalie détectée », soit la liste des anomalies (balise concernée, règle enfreinte, gravité).
  5. Besoin d'une trace ? Cliquez « Export PDF du rapport » Pro.
  6. Retrouvez vos contrôles via « Historique des validations » (date, fichier, rulebook, compteurs — le contenu n'est pas stocké).
  7. Fichier en erreur ? Cliquez « Fichier KO ? Obtiens le diagnostic complet → » pour basculer vers le Diagnostic.

Diagnostiquer Pro

Explique les rejets et lit les relevés. Formats reconnus : pacs.002, pain.002, erreurs EBICS, et relevés camt.052 / 053 / 054, CFONB 120, MT940.

  1. Onglet Analyser → Diagnostiquer (connexion Pro requise).
  2. Chargez le fichier : zone « Glisser ou cliquer · .xml » (ou « Cliquer pour remplacer » pour en changer).
  3. Vue Diagnostic des rejets : chaque opération rejetée devient une carte. Dépliez-la pour voir Description, Causes probables, Actions recommandées, le niveau de Confiance et l'Origine du message.
  4. Si le fichier est un relevé, basculez sur Relevé complet : toutes les écritures s'affichent, avec une recherche (date, montant, tiers, référence) ; les rejets sont surlignés.
  5. Exportez les rejets en CSV ou TXT pour les retravailler.
« Aucun rejet détecté dans ce fichier » = le compte rendu est bon (tout est passé). Utilisez alors la vue Relevé pour l'inventaire complet.

Comparer Gratuit

  1. Onglet Analyser → Comparer.
  2. Chargez le fichier A puis le fichier B.
  3. Les différences structurelles s'affichent — pratique pour vérifier l'impact d'une modification ou l'écart entre deux versions d'un flux.

Pourquoi ? — Contraintes bancaires Pro

  1. Onglet Analyser → Pourquoi ?.
  2. Consultez les contraintes et particularités par banque qui peuvent provoquer un rejet, au-delà de la simple conformité au rulebook.
  3. Croisez avec le Diagnostic pour anticiper un refus avant émission.

Convertisseur d'adresses Pro / 19 €

Convertit les adresses non structurées (lignes libres AdrLine) vers le format structuré ISO 20022 (PstlAdr), obligatoire à l'échéance novembre 2026.

  1. Onglet Analyser → Adresses (ou la tuile de l'Accueil).
  2. Chargez votre fichier. Chaque adresse s'affiche : Adresse originale (AdrLine) à gauche, Aperçu PstlAdr converti à droite.
  3. Repérez les statuts : conforme, déjà conforme ISO 20022, champs manquants obligatoires, ou saisie manuelle requise.
  4. Corrigez si besoin les champs manquants directement dans l'aperçu.
  5. Essai gratuit : un fichier jusqu'à 25 adresses. Au-delà, cliquez « Débloque l'export complet » (achat unique 19 € à vie, ou inclus dans le Pro).
  6. Cliquez « Exporter le fichier converti » pour récupérer le XML corrigé. Retrouvez vos exports dans « Historique des conversions » (sans les adresses).

Générer Gratuit

Produit des fichiers de test conformes (pain.001 / pain.008).

  1. Onglet Analyser → Générer.
  2. Choisissez le type : SCT, SCT Inst, SDD Core, SDD B2B, ICT (trésorerie/VSOT) ou XCT (virement international).
  3. Réglez le Nombre de transactions, les montants et les tiers.
  4. Option avancée : Défauts à injecter pour produire un fichier volontairement non conforme (utile pour tester vos propres contrôles).
  5. Téléchargez le XML généré, prêt à tester (ou à revalider dans Valider).

Référentiels Gratuit

  1. Onglet Référentiels.
  2. Choisissez le sous-onglet : Services bancaires, Balises XML ISO 20022, ou EDIFACT.
  3. Parcourez ou recherchez pour retrouver la signification d'une balise, d'un service ou d'un segment.

EBICS & Motifs de rejet Signification gratuite Résolution Pro

  1. Onglet EBICS pour les erreurs 900xx / 91xxx ; les motifs de rejet ISO (AC01, MS03…) sont accessibles depuis l'Accueil (« Motifs rejets »).
  2. Recherchez un code : sa signification (code, libellé, description) est gratuite.
  3. La résolution (causes probables, actions correctives, qui doit agir) s'affiche pour les abonnés Pro.
  4. Retrouvez aussi les versions EBICS (H003 / H004 / H005) et les protocoles (onglet Plus).

S'abonner & gérer son compte

  1. Sur un écran Pro verrouillé, cochez la case CGV (accès immédiat) — les boutons de prix s'activent.
  2. Choisissez 24 € / mois ou 264 € / an. Pour le seul convertisseur : 19 €.
  3. Le paiement s'ouvre chez Stripe (sécurisé). Renseignez la carte et, pour la facture, votre adresse / n° de société.
  4. Au retour, l'accès Pro est actif immédiatement et vous revenez sur la fonctionnalité.
  5. Une facture est émise automatiquement (mention « TVA non applicable, art. 293 B du CGI »).
  6. Gérez ou résiliez à tout moment via vos initiales → compte ; l'accès reste actif jusqu'à la fin de la période payée.

Bonnes pratiques

  • Vos fichiers ne quittent jamais votre poste : analysez des données réelles en confiance.
  • Autorisez les fenêtres pop-up pour l'export PDF (il ouvre un onglet d'impression).
  • Enchaînez Valider → Diagnostiquer pour passer d'un « KO » à sa résolution.
  • Utilisez Générer avec « Défauts à injecter » pour éprouver vos propres contrôles.
