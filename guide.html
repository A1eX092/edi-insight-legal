Guide utilisateur — EDI Insight

L'outil d'analyse des flux bancaires ISO 20022, EBICS & EDIFACT — 100 % dans votre navigateur.

EDI Insight aide les professionnels des paiements bancaires (trésorerie, monétique, éditeurs, intégrateurs) à valider, diagnostiquer, comparer, convertir et générer des fichiers de paiement, et à consulter les référentiels EBICS / ISO 20022. Prise en main rapide

Qu'est-ce qu'EDI Insight ?

Une application web spécialisée dans les flux de paiement bancaires : virements et prélèvements SEPA (ISO 20022), comptes rendus (pacs.002, pain.002, camt), erreurs EBICS et messages EDIFACT. Elle sert à vérifier la conformité d'un fichier, comprendre un rejet, et préparer/convertir des fichiers — sans installation.

Le principe : analyse 100 % locale

Vos fichiers (montants, IBAN, noms de tiers…) sont traités uniquement dans votre navigateur. Rien n'est envoyé, stocké ni consulté sur nos serveurs. C'est un point clé pour un usage en environnement bancaire sensible.

Les 5 usages clés

  1. Valider un fichier XML contre les rulebooks SEPA (SCT, SCT Inst, SDD Core/B2B…).
  2. Diagnostiquer un fichier de rejet ou lire un relevé (camt) pour comprendre ce qui coince.
  3. Convertir des adresses non structurées vers le format structuré ISO 20022 (échéance nov. 2026).
  4. Générer des fichiers de test conformes (SCT, SDD, ICT, XCT).
  5. Consulter les référentiels : erreurs EBICS 900xx, motifs de rejet (AC01, MS03…), balises XML.

Comment démarrer

  1. Ouvrez app.ediinsight.app (aucune installation).
  2. Les fonctions gratuites sont accessibles immédiatement : Valider, Générer, les référentiels et les significations de codes.
  3. Pour les fonctions Pro (Diagnostiquer, Pourquoi, résolutions, export PDF, convertisseur), connectez-vous avec votre email : vous recevez un code à usage unique à saisir dans l'app.
  4. Glissez-déposez votre fichier, ou cliquez pour le sélectionner. Le résultat s'affiche instantanément.

Guide complet

Détail de chaque fonctionnalité. Les mentions Gratuit et Pro indiquent l'accès requis.

Se connecter

La connexion est sans mot de passe : saisissez votre email, vous recevez un code à usage unique par email, que vous tapez dans l'application. Vous restez sur la page : pas de redirection. Un cookie de session sécurisé maintient votre connexion. Vos coordonnées se limitent à l'email (+ les informations de facturation si vous vous abonnez).

Valider Gratuit

Contrôle la conformité d'un fichier XML de paiement contre les rulebooks officiels (SEPA Credit Transfer, SCT Inst, SEPA Direct Debit Core & B2B…).

  • Chargez le fichier : la liste des anomalies s'affiche (balise concernée, règle, gravité).
  • Un fichier conforme est signalé « OK ». Un fichier en erreur propose un accès direct au Diagnostic.
  • Historique local des validations (date, fichier, rulebook, compteurs) — sans stocker le contenu.
  • Export PDF du rapport de validation Pro.

Diagnostiquer Pro

Analyse les fichiers de rejet et comptes rendus : pacs.002, pain.002, erreurs EBICS (900xx / 91xxx), et lit les relevés camt.052 / 053 / 054, CFONB 120, MT940.

  • Diagnostic des rejets : chaque opération rejetée est expliquée (code, motif, cause, action).
  • Relevé complet : liste de toutes les écritures, avec recherche (date, montant, tiers, référence) et les rejets surlignés.
  • Export des rejets en CSV ou TXT.

Comparer Gratuit

Compare deux fichiers XML pour repérer les différences structurelles — utile pour vérifier l'impact d'une modification ou l'écart entre deux versions d'un même flux.

Pourquoi ? — Contraintes bancaires Pro

Aide à comprendre pourquoi une banque pourrait rejeter un fichier : rappelle les contraintes et particularités par établissement, au-delà de la simple conformité au rulebook.

Convertisseur d'adresses Pro / achat unique 19 €

Convertit les adresses non structurées (lignes libres AdrLine) vers le format structuré ISO 20022 (rue, code postal, ville, pays), rendu obligatoire à l'échéance de novembre 2026.

  • Analyse et aperçu libres ; essai gratuit d'un fichier jusqu'à 25 adresses.
  • Export XML complet après déblocage (achat unique à vie 19 €, ou inclus dans l'abonnement Pro).
  • Journal des exclusions et code couleur selon la qualité de conversion ; historique local (sans les adresses).

Générer Gratuit

Produit des fichiers de test conformes (pain.001 / pain.008) :

  • SCT (virement SEPA) et SCT Inst (virement instantané) ;
  • SDD Core et SDD B2B (prélèvements) ;
  • ICT (trésorerie / VSOT) et XCT (virement international).

Paramétrez le nombre d'opérations, montants et tiers, puis téléchargez un XML valide prêt à tester.

Référentiels Gratuit

Base de consultation : Services bancaires, balises XML ISO 20022, et codes EDIFACT. Pratique pour retrouver la signification d'une balise ou d'un service.

EBICS & Motifs de rejet

Référentiel des erreurs EBICS (900xx / 91xxx) et des motifs de rejet ISO (AC01, MS03…). La signification (code, libellé, description) est gratuite ; la résolution détaillée (causes probables, actions correctives, qui doit agir) est Pro. On y trouve aussi les versions EBICS (H003 / H004 / H005) et les protocoles.

Abonnement & facturation

  • Pro : 24 € / mois ou 264 € / an (un mois offert). Débloque toutes les fonctions Pro.
  • Convertisseur : 19 € en achat unique à vie (inclus dans le Pro).
  • Paiement sécurisé par Stripe ; une facture est émise automatiquement.
  • Résiliation à tout moment depuis votre espace ; l'accès reste actif jusqu'à la fin de la période payée.
  • TVA non applicable, art. 293 B du CGI.

Bonnes pratiques

  • Vos fichiers ne quittent jamais votre poste : vous pouvez analyser des données réelles en confiance.
  • Autorisez les fenêtres pop-up pour l'export PDF (il ouvre un onglet d'impression).
  • En cas de fichier « KO » en validation, utilisez le bouton d'accès direct au Diagnostic.
